## Installation & Usage
1. Add this extension to enable **GOWIN FPGA** support.
2. Set the Path in **Settings -> Tools -> Gowin**
2. Configure your project with `.vhd`, `.vhdl`, `.v`, `.sv`, `.sdc` files.
3. Use the **GOWIN toolchain** for synthesis and programming.
